# 🪑 Vishwakarma Furniture

A modern furniture e-commerce website built with **Django** that allows users to browse furniture categories, explore products, manage their cart and wishlist, and place orders through an elegant and responsive interface.

## 🌐 Live Demo

🔗 https://vishwakarma-furniture.onrender.com

> **Note:** This project is hosted on Render. The first visit may take a few seconds if the server is waking up.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/raj-gajjar610/Vishwakarma-Furniture.git
```

### Move into the project

```bash
cd Vishwakarma-Furniture
```

### Create a virtual environment

```bash
python -m venv .venv
```

### Activate Virtual Environment

**Windows**

```bash
.venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Database Migrations

```bash
python manage.py migrate
```

### Start the Server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
